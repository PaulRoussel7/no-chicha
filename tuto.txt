Bienvenue sur le tutoriel de TextGameEngine

Merci d'avoir choisi text game engine pour vous évader le temps d'un instant.
Ce programme est une interface graphique permettant de créer vos propres histoires.
Les possibilités sont infinies et vous pouvez vous arrêter en pleine conception pour la reprendre plus tard.


Comment créer une histoire :

Pour créer une nouvelle histoire, rien de plus simple. Il vous suffit de cliquer sur le bouton du même nom. Vous allez ensuite arriver sur la page d'édition.

Une histoire est une suite de questions dont les réponses vous emmènent vers d'autres questions. L'Engine vous permet donc de d'inventer vos propres questions, en créant la description d'une situation, et deux réponses, en créant les choix et en les liant à la situation suivante.
Vous avez ensuite la possibilité d'ajouter un monstre que le joueur pourra combattre après avoir sélectionné un choix ou alors un objet qu'il pourra récupérer pour améliorer ses capacités.

Enfin, vous pouvez sauvegarder votre histoire. Vous obtiendrez un fichier .csv que vous pouvez modifier plus tard ou ouvrir dans la section lecteur d'histoire


Comment jouer à un jeux :

Afin de jouer, il suffit de cliquer sur le bouton "jouer à un jeu". L'explorateur de fichier s'ouvre alors pour que vous puissiez sélectionner une histoire


Commandes du jeu :

Clique gauche sur l'un des deux choix que vous voulez faire.
